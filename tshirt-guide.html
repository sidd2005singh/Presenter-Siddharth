3 Step 1: Create a New Replit Project

  1. Go to replit.com and log in
  2. Click the "+ Create" button
  3. Choose template: Select "HTML, CSS, JS" (for pure frontend)
  4. Give your project a name, for example: tshirt-branding-app
  5. Click "Create Repl"
‚úì Success! Now you're inside your Replit workspace.

5 Step 3: Add Basic Structure in index.html

Open index.html and replace the default content with:

index.html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <div class="container">

            <label for="shirtColor">Choose T-Shirt Color:</label>

            <label for="textOverlay">Add Text:</label>
            <input type="text" id="textOverlay" placeholder="Enter your text">
        </div>
        
        <div class="preview-area">
            <div class="tshirt" id="tshirt">
                <img id="logoPreview" style="display:none;" alt="Logo">

    </div>
    
    <script src="script.js"></script>

What you did: Created a title, color picker, file upload, text input, and a preview area for the T-shirt.

7 Step 5: Add Interactivity in script.js

Open script.js and add:

script.js
const shirtColor = document.getElementById("shirtColor");
const logoUpload = document.getElementById("logoUpload");
const textOverlay = document.getElementById("textOverlay");
const textPreview = document.getElementById("textPreview");
const logoPreview = document.getElementById("logoPreview");
const tshirt = document.getElementById("tshirt");

// Change T-shirt color
shirtColor.addEventListener("input", (e) => {
    tshirt.style.backgroundColor = e.target.value;
});

// Update text on T-shirt
textOverlay.addEventListener("input", (e) => {
    textPreview.textContent = e.target.value;
});

// Handle logo upload
logoUpload.addEventListener("change", (e) => {
    const file = e.target.files[0];
    if (!file) return;
    
    const reader = new FileReader();
    reader.onload = (event) => {
        logoPreview.src = event.target.result;
        logoPreview.style.display = "block";
    };
    reader.readAsDataURL(file);
});
What you did: Connected the color picker to the T-shirt background, text input to the preview, and file upload to the logo display.

8 Step 6: Run and Test the App

  1. Click the green "Run" button at the top of the screen
  2. Replit will:
    • Start a web server automatically
    • Open the Web Preview tab on the right
  3. In the preview:
    • Change the color using the color picker
    • Type text and see it appear on the T-shirt
    • Upload an image and see it as a logo on the T-shirt
If something breaks: Check the Console tab for errors, fix the code, and click "Run" again.

10 Step 8: Publish Your App

  1. Make sure your app runs correctly in the Web Preview
  2. Click the "Publish" button in the top-right corner
  3. In the publish panel:
    • Choose "Autoscale deployment" (recommended for production-ready hosting)
    • Click "Deploy"
  4. Replit will:
    • Build and deploy your app
    • Give you a public URL like: tshirt-branding-app.siddharth.repl.co
